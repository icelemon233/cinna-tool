# CinnaTool Todo 使用说明

## 功能概述

CinnaTool Todo 是一个参考 Microsoft To Do 设计的个人任务管理工具，支持多视图、自定义列表、优先级标记、截止日期等功能。

## 视图说明

### ☀️ 我的一天
- 每天手动选择今天要专注完成的任务
- 只显示标记为"我的一天"的任务
- 在此视图添加的任务会自动标记为"我的一天"
- 适合每天早上规划当日待办

### ⭐ 重要
- 显示所有标记为重要（星标）的任务
- 点击任务右侧的 ☆ 可标记/取消重要
- 在此视图添加的任务会自动标记为重要
- 适合追踪高优先级事项

### 📅 计划内
- 显示所有设置了截止日期的任务
- 按截止日期升序排列（最近的在前）
- 过期任务会以红色日期显示
- 在此视图添加的任务会自动设置截止日期为今天
- 适合按时间线管理任务

### 📋 所有任务
- 显示全部任务的平铺列表
- 不做任何过滤
- 适合全局查看和管理

### 自定义列表
- 用户可创建任意数量的分类列表（如"工作"、"生活"、"学习"）
- 每个列表可选择一个 emoji 图标
- 在列表视图添加的任务会自动归入该列表
- 列表可删除（列表内的任务会移到"所有任务"）

## 操作指南

### 添加任务
1. 点击页面底部的 `+ 添加任务` 按钮
2. 输入任务标题
3. 按 Enter 或点击"添加"按钮确认
4. 任务会根据当前视图自动设置对应属性

### 完成/恢复任务
- 点击任务左侧的圆形 checkbox 切换完成状态
- 已完成的任务会折叠在"已完成 N"区域中
- 点击"已完成"可展开/收起已完成任务列表

### 标记为重要
- 点击任务右侧的 ☆ / ⭐ 按钮切换重要标记
- 标记为重要的任务会出现在"⭐ 重要"视图中

### 设置截止日期
- 右键点击任务，选择"设为今天截止"
- 再次右键可选择"清除截止日期"
- 有截止日期的任务会出现在"📅 计划内"视图中

### 添加到"我的一天"
- 右键点击任务，选择"添加到我的一天"
- 再次右键可选择"从我的一天移除"

### 删除任务
- 方式一：鼠标悬停在任务上，点击右侧出现的 🗑️ 按钮
- 方式二：右键点击任务，选择"删除任务"
- 已完成和未完成的任务都可以删除

### 搜索任务
- 在左侧栏顶部的搜索框中输入关键词
- 实时过滤显示标题包含关键词的所有任务
- 清空搜索框恢复正常视图

### 创建自定义列表
1. 点击左侧栏底部的 `+ 新建列表`
2. 点击左侧 emoji 按钮选择图标
3. 输入列表名称
4. 按 Enter 确认创建

### 删除自定义列表
- 鼠标悬停在列表名称上，点击右侧出现的 ✕ 按钮
- 列表删除后，其中的任务不会被删除，会移到"所有任务"中

## 快捷操作

| 操作 | 方式 |
|------|------|
| 添加任务 | 点击底部 `+ 添加任务`，输入后按 Enter |
| 完成任务 | 点击左侧圆形 checkbox |
| 标记重要 | 点击右侧 ☆ 按钮 |
| 删除任务 | hover 后点击 🗑️ 或右键菜单 |
| 搜索 | 在左侧搜索框输入关键词 |
| 切换视图 | 点击左侧导航项 |
| 取消输入 | 按 Esc 键关闭输入框 |
| 右键菜单 | 右键点击任务项 |
